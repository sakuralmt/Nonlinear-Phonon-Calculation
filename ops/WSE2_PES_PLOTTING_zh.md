# WSe2 离线数据分析与绘图工具

本目录中的脚本是独立工具，不接入 `npc`、不接入主流程，也不写运行 contract。它们的作用是基于已经完成的 `WSe2` 正式基线数据，统一产出：

- 定量数据表
- `MAE / RMSE / MaxAE` 柱状图
- `phi122` 对比柱状图
- top1 的 `stage2 vs stage3` 四宫格 3D 对比图
- `GPTFF v1 / v2` 的四阶 vs 五阶重拟合图
- 对应 JSON / Markdown 摘要

## 1. 一键入口

脚本：

- `ops/run_wse2_baseline_analysis.py`

用途：

- 作为唯一的一键入口
- 顺序调用：
  - 数据抽取脚本
  - 柱状图脚本
  - 四宫格 3D 图脚本
  - 高阶重拟合图脚本
- 默认使用当前已经验证过的正式 WSe2 基线数据链

默认输出目录：

- `reports/output/wse2_baseline_analysis/`

默认数据来源：

- 正式 baseline 排名摘要：
  - `remote_baselines/wse2_stage3_run/...`
- `GPTFF v1 / v2` 原始网格：
  - `local_runs/wse2/remote_stage1_rescreen/...`
- `CHGNet` 单-pair 原始网格：
  - 当前归档的正式核心数据副本
- `QE` 原始 top1 网格：
  - 当前归档的真实 `stage3` top1 bundle

## 2. 定量数据与柱状图

脚本：

- `ops/extract_wse2_baseline_metrics.py`
- `ops/plot_wse2_baseline_metrics.py`

用途：

- 抽取前五个候选 pair 的定量数据
- 生成：
  - JSON / CSV / Markdown 表格
  - `phi122` 柱状图
  - 相对 QE 的 `phi122` 误差图
  - `RMSE` 对比图
  - `MAE / RMSE / MaxAE` 总体误差柱状图

## 3. 四宫格模型对照图

脚本：

- `ops/plot_pes_model_vs_qe_2x2.py`

用途：

- 对同一个 `pair_code`
- 分别读取：
  - `GPTFF v1`
  - `GPTFF v2`
  - `CHGNet`
  - 真实 `QE stage3`

四组原始网格数据；

- 对每一组原始网格重新执行同一套当前多项式拟合
- 每个子图同时绘制：
  - 原始数据点
  - 拟合曲面
- 最终输出为 `2x2` 大图

默认目标是用于直接比较：

- 曲面是否围绕原点对称
- 原始点和拟合面的贴合程度
- 不同模型与真实 QE 的差异

当前实现已修正两类容易出错的问题：

- `QE` 侧不再因错误转置导致 `phi122` 被打成接近零
- 同一子图里，原始数据点与拟合曲面的 `A1 / A2` 方向保持严格同向

## 4. 单模型高阶重拟合比较

脚本：

- `ops/plot_pes_refit_compare.py`

用途：

- 对单个模型的同一份原始网格
- 并排比较：
  - 当前四阶拟合
  - 更高阶的五阶拟合

每个子图都同时绘制：

- 原始数据点
- 拟合曲面

同时输出：

- `phi122`
- `r2`
- `rmse`
- 高阶拟合相对于当前拟合的 `phi122` 变化
- 对应 JSON 摘要

## 5. 当前使用原则

这些脚本都按以下原则实现：

- 不直接复用旧 `summary.json` 里的拟合系数作为最终图面
- 而是从原始二维能量网格重新拟合
- 保证不同模型之间比较时使用的是同一套拟合程序

## 6. 适用边界

这些脚本是分析工具，不进入主工作流：

- 不修改 `npc`
- 不修改 `stage2`
- 不修改 `stage3`
- 不写入运行 contract

它们的职责只是在已有数据之上做可视化和拟合稳定性检查。

## 7. 当前应如何解读结果

这批图表和重拟合脚本的目标，是把两类问题拆开看清：

1. 标量三阶耦合系数 `phi122` 是否与 QE 接近
2. 完整二维 PES 曲面是否与原始数据贴合、以及对拟合基底是否敏感

对当前 `WSe2` 正式 baseline 的 top1 pair，正确结论是：

- 在当前四阶拟合下，`GPTFF v1 / v2` 的 `phi122` 与 QE 仍然较接近，这一点与已有定量柱状图结论一致
- 新增的四宫格和高阶重拟合图暴露的是另一层问题：
  - `GPTFF v1 / v2` 的完整 PES 曲面与数据点贴合弱于 `CHGNet / QE`
  - 当强行切换到更高阶多项式时，`phi122` 会出现明显系数重分配

因此，这些脚本不应用来推翻已有的四阶 `phi122` 定量比较；它们的价值在于补充说明：

- “三阶耦合系数接近”不等于“完整 PES 形状已经可靠”
- 高阶重拟合的漂移主要用于诊断 PES 表面稳定性，而不是直接否定四阶 `phi122` 结果
